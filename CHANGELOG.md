# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-10-30

### Added
- Initial open-source release of the project.
- `markdownaggregator` CLI to aggregate Markdown folder trees.
- Python library exposing `aggregate_markdown`.
- Support for manifests, hybrid mode, `@include` directives, and YAML front-matter stripping.
- User documentation and governance files (README, LICENSE, CONTRIBUTING, Code of Conduct).
- Quality tooling configuration (pytest, ruff, mypy, pre-commit).
- Test suite covering the core functionality.
